# Run locally
## Prerequisites
- solana-cli 1.14.*
    - `solana-install init 1.14.28`
- Anchor 0.28.0
    - `avm install 0.28.0 && avm use 0.28.0`

## Build and run the exploits
- `anchor build`
- `npm install`
- `anchor run exploit<n> # replace <n> by the number of the exploit`
